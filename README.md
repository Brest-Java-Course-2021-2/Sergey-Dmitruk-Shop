### Stack technology:
1. Java version: JDK 15
2. Framework: Spring:
    * Spring Boot
    * Spring MVC
    * Spring jdbc
3. DB: H2
4. System Build: Maven
5. System control version: GIT
6. Template engine: thymeleaf
7. Logging: log4j
8. Test: junit, mockito
___

### Requirements
1. JDK 11+
2. Git 2.30.2
3. Apache Maven 3.8.4
